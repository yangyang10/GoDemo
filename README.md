### 项目结构文件夹说明
1、cmd 用于命令行  
2、configs 用于配置文件  
3、deployments 部署脚本，Dockerfile  
4、model 用于模型设计  
5、pkg 用于辅助的库  
6、src 核心逻辑层  
7、main.go 函数入口  
8、Makefile项目构建
